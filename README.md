# asistente-tecnico-privacy
Privacy Policy for Asistente Técnico WhatsApp application

## Privacy Policy

This Privacy Policy describes how the **Asistente Técnico WhatsApp application** collects, uses, and protects user information.

### 1. Information We Collect
The application may process the following information:
- Messages sent by users via WhatsApp
- Phone numbers required to identify conversations
- Basic metadata related to message delivery

No sensitive personal data is intentionally collected or stored.

### 2. Use of Information
The collected information is used exclusively to:
- Provide automated technical assistance
- Respond to user inquiries
- Improve the quality and accuracy of responses

Information is not used for advertising or marketing purposes.

### 3. Data Storage
Messages and data may be temporarily processed to generate responses.
No long-term storage of personal data is performed unless strictly necessary for technical operation.

### 4. Data Sharing
User data is **not shared** with third parties, except when required to operate the WhatsApp Business API provided by Meta.

### 5. Security
Reasonable technical measures are applied to protect information against unauthorized access, alteration, or disclosure.

### 6. User Rights
Users may stop interacting with the application at any time by discontinuing WhatsApp communication.
